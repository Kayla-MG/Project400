# Project400
This is my final year project where I am building an integrated supportive app for neurodivergent teens / adolescents.
This is a specialized assistive mobile application designed to facilitate emotional regulation and routine management for neurodivergent individuals. 
Developed with a focus on accessibility and low cognitive load, the application integrates proactive mood tracking with reactive de-escalation strategies to support daily functioning and mitigate crisis situations. 
The interface prioritizes visual-first navigation to reduce verbal demand and cognitive overhead for the user
The Frontend is React Native --> Backend is Node.js & Express --> Database is MySQL (via XAMPP)
